# 🔗 AmalLinks - URL Shortener

A modern, feature-rich URL shortener with custom aliases, QR codes, analytics, and admin dashboard.

## ✨ Features

- 🎯 Custom Aliases
- 📊 Click Analytics
- 📱 QR Code Generation
- ⏰ Expiry Dates
- 🌙 Dark/Light Mode
- 🛡️ Admin Dashboard
- 🎨 Glassmorphism UI
- 📋 One-Click Copy
